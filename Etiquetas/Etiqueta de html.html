Etiqueta de imagen

La etiqueta de imagen permite insertar una imagen dentro de un documento html, la
etiqueta se escribe como "< img >". Esta etiqueta crea un espacio de retención en el
que la imagen referenciada aparecerá más tarde. Dado que se considera un elemento
vacío, no hay etiqueta de cierre.

El "Src" se usa para especificar la ruta del archivo de imagen. Cuando se carga una
página web, el navegador la recupera de un servidor web y la inserta en la página.
Si este atributo no es correcto, el navegador no podrá encontrar la imagen ni
cargarla correctamente.

El atributo "Alt" es responsable del texto alternativo de la foto. Debe describir
con precisión el archivo de imagen porque es lo que se va a mostrar en caso de que
no se cargue la imagen.

Existen atributos para modificar el tamaño de la imagen como "width" y "height".




Etiqueta de audio

Al igual que la etiqueta de imagen, la etiqueta de audio escrita como "< audio >"
tampoco tiene un cierre.

También contiene un atributo para indicar de donde viene, en este caso el "Src"

La etiqueta de audio contiene varios atributos para manipular el archivo cargado.
Como por ejemplo el atributo "mediagroup" sirve para establecer el nombre del grupo
de contenidos multimedia, el "autoplay" como su nombre indica hace que se reproduzca
el audio de manera automática "loop" sirve para que se reproduzca el archivo en bucle
al terminar, "muted" hace que el audio aparezca silanciado por defecto y "controls"
muestra los controles básicos para el manejo el archivo de audio.
